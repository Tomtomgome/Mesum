Detailed Description

template<typename t_RetType, typename... t_Args>
class m::mCallback< t_RetType, t_Args >

Class encapsulating a callback.

It can hold any kind of function (members, lambdas, etc...). Warning, it is not a performant implementation, and there is a std::function under the hood

Template Parameters
t_RetTypeThe return type of the callback
t_ArgsThe list of arguments of the function
