\subsubsection*{template$<$typename... t\+\_\+\+Args$>$\newline
class m\+::m\+Signal$<$ t\+\_\+\+Args $>$}
Class encapsulating a signal. 

Functions registered to a signal can only return void. When a signal is called with arguments, all its registered callback are called. Order is not guaranteed.
